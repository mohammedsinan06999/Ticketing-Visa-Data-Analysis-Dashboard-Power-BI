# Ticketing & Visa Data Analysis using Power BI

## Overview
This project focuses on analyzing ticketing and visa operations data to identify key trends, evaluate salesman performance, and generate business insights. The analysis was performed using Power BI as part of an internship project.

## Objective
- Analyze sales performance across ticketing and visa services  
- Evaluate salesman contribution to total revenue  
- Identify key suppliers and their impact on visa sales  
- Understand transaction patterns and operational status  

## Dataset
The dataset includes:
- Ticketing transactions  
- Visa processing records  
- Salesman details  
- Supplier information  
- Transaction status  

## Data Preparation
- Combined multiple monthly datasets into unified tables  
- Handled missing values and corrected data types  
- Cleaned inconsistent entries and removed errors  
- Created relationships between tables using a data model  

## Data Model
- Salesman (dimension table)  
- Tickets table  
- Visa table  
- Relationships created using Salesman as a common dimension  

## Key Measures (DAX)
- Total Sales  
- Total Profit  
- Total Transactions  
- Average Sale  
- Ticket Sales and Visa Sales  
- Top Performer  

## Dashboard Features

### Page 1: Executive Overview
- KPI Cards (Total Sales, Profit, Transactions, Average Sale)  
- Sales Trend over Time  
- Top 10 Salesmen by Sales  
- Revenue Distribution (Ticket vs Visa)  

### Page 2: Sales & Operational Analysis
- Profit by Salesman  
- Supplier Performance (Visa Sales)  
- Transaction Status Distribution  
- Detailed Transaction Table  

## Key Insights
- Ticketing services contribute the majority of total revenue  
- A small group of salesmen generates a significant portion of sales  
- HADAF is the leading supplier in visa processing  
- Most transactions are cleared, indicating efficient operations  
- Some transactions show negative profit, requiring further review  

## Challenges Faced
- Data cleaning due to missing values and inconsistent formats  
- Combining multiple datasets into a single model  
- Handling date-related issues and errors  
- Creating accurate profit calculations  

## Conclusion
The analysis highlights that ticketing is the primary revenue driver, while visa services provide additional support to business operations. Sales performance varies across salesmen, and operational insights such as transaction status help monitor workflow efficiency.
